数据质量评估

通过多维度评估数据质量，确保训练数据的高质量。

评估维度

准确性、相关性、多样性、平衡性、一致性。

评估方法

人工审核、自动检查、统计分析。

应用场景

数据准备、持续监控、质量改进。

工具

质量评分工具、统计分析工具。

⚙️ 评估维度

  • 准确性：内容正确性、语法正确性、逻辑一致性
  • 相关性：与任务的相关性、内容主题匹配
  • 多样性：内容多样性、表达多样性、主题多样性
  • 平衡性：类别平衡、长度分布、难度分布
  • 一致性：格式一致性、标注一致性

质量评分

def evaluate_quality(example):
    scores = {
        "length": len(example["text"].split()),
        "diversity": calculate_diversity(example["text"]),
        "relevance": calculate_relevance(example["text"]),
    }
    return scores

def calculate_diversity(text):
    words = text.split()
    unique_words = set(words)
    return len(unique_words) / len(words) if words else 0
